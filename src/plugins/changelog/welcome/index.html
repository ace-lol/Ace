Welcome to Ace!
Welcome to Ace! Ace is an extension framework for the new League client that allows developers to create various tweaks and plugins that either change or improve the workings of the client.
Ace is currently in beta. While we try to make sure that every tweak works as expected, you may run into issues. We have implemented a system that tries to minimize unexpected situations, especially when the client updates.

Ace will automatically check for updates whenever you launch the League client via the Ace launcher. If an update is found, it is downloaded in the background and you'll be prompted to restart the client once it is ready. Upon starting the updated client, a small changelog screen will await you with an overview of all changes.

Ace has various tweaks installed, most notably:
  • hide-mobile, for hiding friends on mobile.
  • resize, for resizing the client (see that caret in the bottom right?).
  • owned-skins, which adds a new 'Skins' tab in your collections.

The settings plugin provides various statistics and options, similar to the built-in settings panel. You can access the Ace settings by clicking on the Ace icon in the top right corner. You can also disable or enable plugins here.
